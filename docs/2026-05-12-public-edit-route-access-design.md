# 公开作品编辑页访问分流设计

## 背景

公开作品的游玩页允许所有访问者查看和试玩，但编辑页只属于作品作者。当前非作者访问公开作品 `/games/:id/edit` 会进入 404，用户容易误判为作品不存在或公开状态失效。

## 决策

保持编辑权限不变：只有作品作者可以进入 `/games/:id/edit`。当访问者不是作者，但作品是公开且可玩的 `PUBLIC READY` 状态时，编辑页路由自动跳转到公开游玩页 `/games/:id`。

## 用户影响

- 非作者从错误的编辑链接进入时，会直接看到可玩的公开作品。
- 作者仍然可以进入编辑台继续修改和调整公开状态。
- 私密作品、未完成作品、不可玩作品继续不可见，避免泄露创作内容或半成品。

## 验证

- 公开可玩作品的非作者 `/edit` 访问应跳转到 `/games/:id`。
- 作者访问 `/edit` 应保持编辑台。
- 私密或不存在作品访问 `/edit` 应保持不可见。
