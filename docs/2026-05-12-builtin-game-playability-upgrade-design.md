# 内置游戏可玩性优化设计

## 背景

内置游戏承担 Gallery 首屏体验和新用户理解产品能力的作用。当前 21 个内置游戏共用 `public/builtin-games/shared/engine.js`，多数游戏只是不同配置驱动的轻量玩法，第一局反馈、目标压力、风险回报和美术记忆点不足。

## 目标

- Feed 前 5 个内置游戏做成精品：星港闪避、霓虹砖块、像素守门员、熔岩跳台、深空清道夫。
- 其他内置游戏通过共享引擎平均变好，提升开局理解、反馈密度和难度曲线。
- 星港闪避增加专属游戏美术资产，并在游戏内使用，而不是只替换封面。

## 范围

- 主要修改 `public/builtin-games/shared/engine.js`。
- 星港闪避新增项目内游戏背景资产：`public/builtin-games/starport-dash/starport-bg.png`。
- 必要时同步内置游戏文案，但不改变 Gallery、详情页、我的作品、生成链路和账号范围。

## 精品化设计

### 星港闪避

- 使用生成的星港背景强化游戏内视觉识别。
- 增加能量芯片连击、擦弹得分、临时护盾/加速道具。
- 加入来袭危险预警，让失误更像玩家选择，而不是随机撞上。

### 霓虹砖块

- 增加特殊砖块和连锁爆破，减少基础打砖块的重复感。
- 强化挡板反弹角度，让玩家能主动控球。
- 随进度提升球速和得分倍率。

### 像素守门员

- 增加射门预判线、弧线球和连续扑救倍率。
- 扑救成功时给出更明显的粒子、震动和得分反馈。

### 熔岩跳台

- 优化平台生成节奏，降低不可控断崖。
- 强化下沉平台反馈、跳跃缓冲、落地反馈。
- 冷却晶石不只加分，也延长时间或形成短期目标。

### 深空清道夫

- 与星港闪避区分：突出“拖废件回收”的任务感。
- 满载后移动变慢，回收区连投给额外奖励。
- 陨石加入预警，提高可读性。

## 平均提升设计

- 开局提示展示目标和操作，降低第一次玩的理解成本。
- 街机类加入动态难度和连击奖励。
- 鼠标点击类提高命中容错，减少“点到了但没反应”的挫败。
- 防守/塔防类强化波次、金币、血量反馈。
- 解谜类强化进度、撤销/重试和错误反馈。

## 验证

- 基础验证：`npm run lint`、`npm run build`、`npx prisma generate`。
- 游戏验证：前 5 个全测，其他抽测跑酷、塔防、弹幕、连线、记忆。
- 核心标准：页面可加载、无 fatal JS error、开始/点击/键盘输入能让状态变化。
